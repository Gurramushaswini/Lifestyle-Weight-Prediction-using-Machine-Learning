# 🏋️ Lifestyle Weight Prediction using Machine Learning

## 📌 Project Overview

This project focuses on predicting an individual's **Current Weight (kg)** using machine learning techniques based on lifestyle and nutritional factors. It demonstrates a complete end-to-end machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model development, and performance evaluation using a regression algorithm.

The project highlights how data-driven approaches can be used to estimate body weight based on health and dietary attributes.

---

## 🎯 Problem Statement

The objective of this project is to build a regression model capable of predicting an individual's **Current Weight (kg)** using lifestyle and nutritional information. Accurate weight prediction can support health monitoring and provide insights into the relationship between dietary habits and body weight.

---

## 📂 Dataset

The dataset contains information related to lifestyle, nutrition, and body measurements, including:

- Initial Weight
- Current Weight
- Weight Change
- Daily Calorie Intake
- Protein Intake
- Carbohydrate Intake
- Fat Intake
- Lifestyle-related attributes
